Welcome to GladLogs

About GladLogs

GladLogs is a simple application that collects and stores twitch messages from various channels.

For Streamers

If you are a streamer, and you want to be included in the list of channels that I collect messages from, please contact me at Gladlogs@assasinos.live.

For Developers

If you are a developer, and you want to contribute to the project, you can find the source code on GitHub.

Compliance with EU Regulations

I comply with EU rules and regulations. All the data I store is accessible on this page, and you can request its removal at any time. If you have any concerns or wish to make a request, please feel free to contact me directly via Gladlogs@assasinos.live.
